# Bug 记录 - 2026-03-12

## 问题描述
用户确认发布的版本与实际发布的内容不一致。

## 原因分析
- `confirm-send.js` 通过环境变量传递 `POST_CONTENT` 给 `post-api.js`
- 但 `post-api.js` 没有读取 `POST_CONTENT` 环境变量
- `post-api.js` 直接读取 `today-article.json` 文件
- 该文件包含的是旧的草稿内容

## 修复方案
修改 `confirm-send.js`，在调用 `post-api.js` 前先把正确内容写入 `today-article.json`
